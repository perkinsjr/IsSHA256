# isSHA256

Check to see if something is SHA256 

## Install

    $ npm install --save issha256

## Usage

    const issha256 = require('issha256');

    issha256.test('0c67b2c1fcfba9fb71ff6feae41abd61d591df8ecabebefdba51db4d3cf9f1cf');
    issha256.test('NOT256');
